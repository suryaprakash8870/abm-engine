## Slide 1 — Title
**ABM Engine — What We Built**
The code-native version of the OneGTM Allbound Engine · 11 engines, one event bus

## Slide 2 — The system
Define who → find them → enrich + qualify → score + tier → target list → buying committee → capture intent → score awareness → fire the play → write to CRM → learn from wins. Eleven engines, event-driven, Next.js + Postgres + Ollama, live on Render.

## Slide 3 — Build status
Live (2): TAL Manager · Awareness. Partial (8): ICP · TAM · Scoring · Contacts · Signals · Orchestrator · CRM Sync · Flywheel. Mock (1): Enrichment.
Message: the orchestration, events, gates and APIs are all real; the mock parts are external data feeds.
